\((1) ax^2 + bx + c = 0(a\neq0)\)  的判別式  \(D = b^2 - 4ac\)

   \(\begin{cases} 若 D > 0 \Rightarrow 兩根為相異實數 \\ 若 D = 0 \Rightarrow 兩根為相等實數 \\ 若 D < 0 \Rightarrow 方程式沒有實根 \\ 若 D \geq 0 \Rightarrow 兩根為實數 \end{cases}\)

\((2)\) 根與係數的關係：
       設  \(\alpha\)、\(\beta\) 為二次方程式 \(ax^2 + bx + c = 0\) 的兩根，
       則 \(\alpha + \beta = - \frac{b}{a}，\alpha\beta = \frac{c}{a}\)

\((3)\) 以\(\alpha\)、\(\beta\) 為兩根的方程式 \(\Rightarrow x^2 - (\alpha + \beta)x + \alpha\beta = 0\)
