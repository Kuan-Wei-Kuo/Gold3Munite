\(\Delta{ODC}\) 中的 \(\overline{OC}\) = 8 \(\times\cos30^\circ\) = \(\sqrt[4]{3}\)
\(\Delta{OCB}\) 中的 \(\overline{OB}\) = \(\overline{OC}\times\cos15^\circ\) = \(\sqrt[4]{3}\times\cos15^\circ\)

\(\Delta{OBA}\) 中的 \(\overline{AB}\)

= \(\overline{OB}\times\sin15^\circ\)
= \(\sqrt[4]{3}\times\cos15^\circ\times\sin15^\circ\)
= \(\sqrt[4]{3}\times\frac{1}{2}\sin30^\circ\)
= \(\sqrt[2]{3}\times\frac{1}{2}\) = \(\sqrt{3}\)
